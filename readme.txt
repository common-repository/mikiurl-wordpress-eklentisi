=== Mikiurl Wordpress Eklentisi ===
Tags: mikiurl, link, tiny, kisaltma, url, twitter, posting, shortining
Requires at least: 2.7.1

Yayınladığınız yazılarınızı, Mikiurl ile linklerini kısaltarak Twitter hesabınıza gönderir.

== Description ==

Bu eklenti yeni bir yazı gönderdiğinizde bu yazınızı sizin twitter hesabınızda yayınlar ve yazı linkinizi http://mikiurl.com link kısaltma servisi aracılığıyla kısaltarak paylaşmanızı sağlar. 

== Installation ==
1. İndirdiğiniz eklentiyi Wordpress `/wp-content/plugins/` dizinine gönderin.
2. Daha sonra `Mikiurl Wordpress Eklentisi`ni 'Eklentiler' menüsünden aktif edin.
3. Mikiurl ayar sayfasına giderek (Options > Mikiurl) Twitter üyelik bilgilerinizi giriniz.

in English
1.After downloading, unzip file and upload the enclosed mikiurl directory to your WordPress plugins directory: /wp-content/plugins/.
2.As always, activate the Mikiurl Wordpress plugin on your WordPress plugins page.
3.Go to the Mikiurl Options page (Options > Mikiurl) to set your Twitter account information and preferences.

== Frequently Asked Questions ==

= Son yayınladığımız yazının Twitter hesabımızda görüntülenmesi ne kadar zaman alır? =

Bu zaman Sitenizin ve twitter'ın sunucusunun hızına bağlı olarak değişmekle birlikte 5 ile 15 sn. arasında gerçekleşir.

= Eklentinin çalışması için herhangi bir kod eklemem gerekiyor mu? =

Hayır. Eklentiyi aktif hale getirdikten sonra, Mikiurl Eklenti Ayarlarından, twitter hesabınıza ait bilgileri girdikten sonra eklenti çalışmaya başlayacaktır.

Eklenti hakkında her türlü sorularınız için http://www.autocadhocasi.com/mikiurl-wordpress-eklentisi.html
